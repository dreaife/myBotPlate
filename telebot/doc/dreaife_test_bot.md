## 💡 Bot 介绍

Lagos 是一个功能强大的 AI 聊天助手，基于 Telegram 平台运行。它集成了先进的 AI 技术，能够与用户进行自然、流畅的对话，回答问题、提供建议，并为用户提供智能化的交互体验。

### 核心特性

- 🧠 **智能对话** - 基于 OpenAI GPT 模型，理解上下文并提供准确回复
- 💬 **上下文记忆** - 记住对话历史，实现连贯的多轮对话
- 🔄 **实时响应** - 快速处理消息，显示"正在思考..."状态
- 🗑️ **历史管理** - 支持随时清除对话历史，开始新话题
- 📊 **日志记录** - 完整的日志系统，方便问题追踪和分析
- 🔐 **安全可靠** - 基于环境变量管理敏感信息，确保数据安全

## 🎯 功能说明

### 智能对话

直接向 bot 发送任何消息，它会：
1. 显示"正在思考..."提示
2. 调用 AI 服务处理你的问题
3. 返回智能回复

**示例对话：**
```
你: 你好，请介绍一下自己
Bot: 你好！我是 AI 助手机器人，基于先进的人工智能技术...

你: Python 中如何读取文件？
Bot: 在 Python 中读取文件有多种方式...
```

### 上下文管理

Bot 会自动记住你的对话历史，实现连贯的多轮对话：

```
你: 我想学习 Python
Bot: 很好！Python 是一门优秀的编程语言...

你: 从哪里开始比较好？
Bot: 基于你刚才提到想学习 Python，我建议从以下几个方面开始...
```

## 📋 操作指令

### 基础命令

| 命令 | 功能 | 说明 |
|------|------|------|
| `/start` | 开始对话 | 初次使用时发送，显示欢迎信息 |
| `/help` | 显示帮助 | 查看所有可用命令和使用说明 |
| `/clear` | 清除历史 | 清除当前用户的对话历史，开始新话题 |

### 命令详解

#### `/start` - 开始对话

**功能：** 初始化与 bot 的对话，显示欢迎信息

**使用场景：**
- 第一次使用 bot
- 长时间未使用后重新开始
- 想看到欢迎信息

**示例：**
```
你: /start
Bot: 你好！我是AI助手机器人。请发送消息与我聊天。
```

---

#### `/help` - 显示帮助信息

**功能：** 显示所有可用命令和使用说明

**使用场景：**
- 忘记了有哪些命令
- 想了解 bot 的功能
- 需要查看使用说明

**示例：**
```
你: /help
Bot: 可用命令：
     /start - 开始对话
     /help - 显示帮助信息
     /clear - 清除聊天历史
     
     直接发送消息与AI助手对话。
```

---

#### `/clear` - 清除聊天历史

**功能：** 清除当前用户的对话历史记录

**使用场景：**
- 想开始新的话题
- 对话内容过多，想重新开始
- 需要 bot "忘记"之前的对话内容

**注意事项：**
- ⚠️ 此操作不可撤销
- ⚠️ 只清除你自己的历史，不影响其他用户
- ⚠️ 清除后，bot 将无法引用之前的对话内容

**示例：**
```
你: /clear
Bot: 已清除您的聊天历史。

[现在可以开始全新的对话]
```

## 💡 使用技巧

### 1. 明确的问题

✅ **好的提问：**
```
请用 Python 写一个读取 CSV 文件的示例代码
```

❌ **不太好的提问：**
```
代码
```

### 2. 提供上下文

利用 bot 的上下文记忆功能，进行多轮对话：

```
你: 我想用 Python 做数据分析
Bot: [回答...]

你: 需要学习哪些库？
Bot: [基于前面的对话，知道你在问 Python 数据分析的库...]
```

### 3. 合理使用 /clear

在以下情况考虑清除历史：
- 切换完全不同的话题
- 对话历史过长影响性能
- 需要 bot "忘记"敏感信息

### 4. 查看状态提示

Bot 会显示"正在思考..."提示，表明正在处理你的消息，请耐心等待。

## 🔧 技术特性

### 日志系统

- 📝 每天自动轮转日志文件

### AI 服务

- 🤖 默认使用 OpenAI GPT 模型

## ❓ 常见问题

### Q: Bot 没有回复怎么办？

**A:** 请检查：
1. Bot 是否显示"正在思考..."（表示正在处理）
2. 网络连接是否正常
3. 是否发送了纯文本消息（不支持图片、文件等）

### Q: 如何让 Bot "忘记"之前的对话？

**A:** 发送 `/clear` 命令清除聊天历史

### Q: Bot 支持哪些语言？

**A:** Bot 支持多种语言，包括中文、英文等，会根据你的输入语言自动调整回复语言

### Q: Bot 会保存我的对话吗？

**A:** 
- 对话历史仅在当前会话中保留
- 用于维持上下文连贯性
- 可随时使用 `/clear` 清除
- 日志文件保留 30 天用于系统维护

### Q: 可以同时和多个用户对话吗？

**A:** 可以！每个用户的对话都是独立的，互不影响
